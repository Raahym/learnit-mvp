# LearnIt PRD

## Product Promise

LearnIt helps a student know what to study next, why it matters, and how ready they are for an exam.

## Target User

Students preparing for exams who have scattered notes, weak topics, limited time, and no reliable system for turning study activity into a plan.

## MVP User Journey

1. Sign up and confirm email.
2. Complete onboarding with subjects, exam dates, target grades, and availability.
3. Add topics or weak areas.
4. Upload study material.
5. Take a diagnostic quiz.
6. Review mistakes.
7. Complete generated study tasks.
8. See readiness improve with an explanation.

## Non-Negotiables

- Never invent user data.
- Never show a confident recommendation without evidence.
- Every dashboard recommendation must explain its source.
- Empty states must be useful setup states.
- AI output must be structured and validated before saving.

## MVP Success Criteria

- A new user can create a real study workspace.
- The dashboard reflects only saved user data.
- A user can complete the loop: subject -> topic -> quiz -> mistake -> task -> readiness update.
- Uploads are validated and stored safely.
- The app has basic operational docs and security controls.
